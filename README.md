## **Descripcion**:
#### How does it work?
Con esta aplicación podrás escuchar la musica de la que disponemos. La página esta compuesta de:
*  Temperatura Actual
*  Temperatura máxima
*  Temperatura mínima
*  Humedad

Adicionalmente, al seleccionar una ciudad te mostrará una imagen de ella para hacer más atractiva la web.
